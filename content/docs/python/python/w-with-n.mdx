## Integers

Integers are numbers without any fractional part and can be positive (1, 2, 3, ...), negative (-1, -2, -3, ...), or zero (0).

```py
age = 25
print(age)
```

```
25
```

You can use the `type()` function to determine the data type of a value or variable.

```py
age = 25
print(type(age))
```

```py
<class 'int'>
```

In the output above, `<class 'int'>` refers to the **int**eger data type

## Float

Floating-point numbers are numbers with a decimal point, representing fractional values.

```py
pi = 3.14159265
print(pi)
print(type(pi))
```

```
3.14159265
<class 'float'>
```

## Arithmetic Operations

You can perform arithmetic operations on numbers using operators like `+`, `-`, `*`, `/`, `//` (integer division), `%` (modulus), and `**` (exponentiation).

```py
a = 10
b = 3

print(a + b)
```

```sh
13
```

| Operator | Name           | Description                                         |
| -------- | -------------- | --------------------------------------------------- |
| `a + b`  | Addition       | Sum of `a` and `b`                                  |
| `a - b`  | Subtraction    | Difference of `a` and `b`                           |
| `a * b`  | Multiplication | Product of `a` and `b`                              |
| `a / b`  | True division  | Quotient of `a` and `b`                             |
| `a // b` | Floor division | Quotient of `a` and `b` , removing fractional parts |
| `a % b`  | Modulus        | Integer remainder after division of `a` and `b`     |
| `a ** b` | Exponentiation | `a` raised to the power of `b`                      |
| `-a`     | Negation       | The negative of `a`                                 |

_Example:_

```py
a = 10
b = 3

print(a + b)  # Output: 13 (Addition)

print(a - b)  # Output: 7 (Subtraction)

print(a * b)  # Output: 30 (Multiplication)

print(a / b)  # Output: 3.3333333333333335 (Division)

print(a // b)  # Output: 3 (Integer division)

print(a % b)  # Output: 1 (Modulus)

print(a ** b)  # Output: 1000 (Exponentiation)
```

## Order of Operations

Python follows the standard order of operations (**PEMDAS**: **P**arentheses, **E**xponents, **M**ultiplication/**D**ivision, **A**ddition/**S**ubtraction) when evaluating arithmetic expressions.

```py
result = 9 - 6 + 3
result = -5 + 7 * 2
```

```sh
6
9
```

You can use parentheses to override the default order of operations.

```py
result = (9 - 6) + 3
result = -5 + (7 * 2)
```

```sh
6
9
```

## Builtin functions

Python also provides built-in functions to work with numbers, here are the most important functions:

| Operator    | Name           | Description                                    |
| ----------- | -------------- | ---------------------------------------------- |
| `abs(x)`    | Absolute Value | Returns the absolute value of `x`              |
| `round(x)`  | Rounding       | Rounds the value of `x` to the nearest integer |
| `max(a, b)` | Maximum Value  | Returns the larger value between `a` and `b`   |
| `min(a, b)` | Minimum Value  | Returns the smaller value between `a` and `b`  |
| `type(x)`   | Type           | Returns the type of the object `x`             |

**Example:**

```py
print(abs(-5))  # Output: 5

print(round(3.14159))  # Output: 3

print(max(10, 20))  # Output: 20

print(min(10, 20))  # Output: 10
```

You can also specify the number of decimal places for rounding using the `round()` function.

```py
print(round(3.14159, 2))
```

```
3.14
```

You can also compare more than two numbers using the `max()` and `min()` functions.

```py
print(max(10, 20, 5))
print(min(10, 20, 5))
```

```
20
5
```
